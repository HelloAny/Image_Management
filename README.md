eggjs静态图片管理工具
学习egg

## 总结

最开始使用iview组件，但是路上遇到了不少坑，再加上我感觉iview组件的完善程度不如element，于是用element组件重新做了一遍，虽然最终想要达到的效果不如我一开始设想的，但是这也算eggjs的第一个练手项目，之后我会不断改善这个项目
